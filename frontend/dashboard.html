SmartApply Chrome Extension

Apply to jobs directly from your browser. No server bot needed.

🧩 Download Extension

Setup Instructions

1
Download & unzip the extension
Click "Download Extension" above and unzip the file to a permanent folder on your computer.
2
Load in Chrome
Go to chrome://extensions → Enable Developer Mode → Load Unpacked → select the unzipped folder.
3
Sign in with your token
Click "Copy Auth Token" above. In the extension popup, paste it as your token along with this server's URL.
4
Start applying on LinkedIn
Visit LinkedIn Jobs, open the extension popup, and click Start Bot. Applications will log here automatically.
